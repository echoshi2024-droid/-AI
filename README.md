# 🥬 买菜记账

一个手机版网页工具，通过上传微信转账截图自动统计买菜金额。

## ✨ 功能特点

- 📱 **手机版优化**：完美适配 iPhone 屏幕
- 🖼️ **多图上传**：一次可上传多张截图
- 🔍 **自动识别**：使用 Tesseract.js OCR 识别金额
- 💰 **自动累加**：智能提取并计算总额
- 🌐 **无需服务器**：纯前端运行，隐私安全

## 🚀 使用方法

### 方式一：本地直接打开
1. 在文件管理器中找到 `index.html`
2. 双击用浏览器打开（推荐 Safari 或 Chrome）
3. 上传微信转账截图
4. 点击"开始识别"
5. 查看统计结果

### 方式二：部署到网络（可选）
可以使用以下免费服务部署：
- **Netlify**: 拖拽文件夹到 netlify.com
- **Vercel**: 连接 GitHub 仓库自动部署
- **GitHub Pages**: 推送到 GitHub 启用 Pages

## 💡 使用技巧

1. **截图要清晰**：确保转账金额部分清晰可见
2. **光线充足**：避免反光和阴影
3. **完整截图**：包含完整的转账信息区域
4. **多张累加**：可以一次上传多张，自动累加所有金额

## 🔧 技术说明

- **OCR 引擎**: Tesseract.js v5
- **语言支持**: 简体中文
- **浏览器要求**: 支持 ES6 的现代浏览器
- **网络要求**: 首次加载需要网络（加载 Tesseract.js 和语言包）

## ⚠️ 注意事项

- 首次使用需要下载 OCR 语言包（约 20MB），请保持网络连接
- 识别准确率取决于截图清晰度
- 建议在光线好的环境下使用
- 识别结果请人工核对
